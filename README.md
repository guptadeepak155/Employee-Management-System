Employee Management System

Employee Management System is a full-stack platform built with the MERN stack (MongoDB, Express.js, React.js, Node.js) to simplify HR tasks. It allows tracking attendance, managing leaves, monitoring performance, and generating reports efficiently. The system is secure, user-friendly, and designed to help organizations manage their workforce effectively.

Features

Add, update, and delete employee records

Track attendance and manage leaves

Monitor employee performance

Generate detailed reports

Secure role-based access for employees and admins

Dashboard-style UI for administrators

Automated notifications

Tech Stack

Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB

Authentication: Secure login with role-based access

Highlights

Developed a full-stack platform to manage employee records, departmental grouping, and onboarding workflows

Implemented CRUD operations, secure authentication, and form validations for accurate data entry

Integrated REST APIs with MongoDB for real-time data handling and scalability

Created a clean dashboard-style UI for administrators to view, filter, and update employee information

Enabled role-based access control to ensure secure and structured system usage
